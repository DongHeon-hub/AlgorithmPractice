다이나믹 프로그래밍(Dynamic Programming, DP)
: 여러 개의 하위 문제를 먼저 푼 후 그 결과를 쌓아올려 주어진 문제를 해결하는 알고리즘

-> 문제를 해결하기 위한 점화식을 찾아낸 후, 항을 밑에서부터 차례로 구해나가서 답을 구함


가장 쉬운 예시로는 피보나치 수열이 있다.
중간 중간 결과를 저장해 나가면서, 여러개의 하위 문제를 푼 후 그 결과를 쌓아올려 최종적인 결과를 구한다.
코드는 다음과 같다.

int fibo(int n){
    int fib[20];
    fib[0]=fib[1]=1;
    
    for(int i=2; i<=n; i++) fib[i] = fib[i-1]+fib[i-2];

    return fib[n];
}

* DP를 푸는 과정

1. 테이블 정의
2. 점화식 찾기
3. 초기값 정하기
