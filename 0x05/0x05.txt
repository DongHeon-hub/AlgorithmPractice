스택

한쪽 끝에서만 원소를 넣거나 뺄 수 있는 자료구조
Restricted structure 라고 부르기도함

스택의 성질
1. 원소의 추가 : O(1)
2. 원소의 제거 : O(1)
3. 제일 상단의 원소 확인 : O(1)
4. 제일 상단이 아닌 나머지 원소들의 확인/변경이 원칙적으로 불가능
기본적으로 원소의 추가/원소의 제거/제일 상단의 원소 확인만을 필요로 함

구현. 배열 or 연결 리스트를 이용해 구현할 수 있음


const int MX =1000005;
ind dat[MX];
int pos=0;


stack 의 구현은 정말 쉽지만, stl의 stack을 쓸 수 있다면 쓰는 것이 좋다.
의심할 필요가 없기 때문
